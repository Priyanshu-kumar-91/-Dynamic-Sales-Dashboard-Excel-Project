1. Structured Three-Panel Layout

The dashboard is designed with a clean three-panel layout—slicers on the left for filtering,
KPI cards at the top for quick insights, and all visual charts positioned on the right—creating a smooth and intuitive data exploration flow.

2. Delivery Status Color Coding

A clear and consistent color palette is applied for delivery statuses—Green (Delivered), 
Yellow (In-Transit), Orange (Returned), and Red (Cancelled)—making each status instantly distinguishable and visually easy to interpret.

3. Prominent KPI Cards

Key metrics such as Online and Offline Sales are showcased in bold KPI cards with highly visible numeric values, 
enabling users to capture important performance indicators at a glance without scrolling.

4. Logically Organized Visuals

The visual section includes a combination of pie charts, donut charts, bar charts, city-wise line charts,
and monthly performance bars—arranged logically to ensure clear comparisons and smooth insight discovery.

5. User-Friendly Slicer Panel

The left-side slicer panel includes filters for Date, Category, Sub-Category, Region, and City.
These neatly arranged slicers allow users to apply multiple filters effortlessly while keeping the dashboard layout clean and uninterrupted.
